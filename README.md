# 準備
## fork
本リポジトリをforkしてください。

## java
java11をインストールしてください。

## import project
IDEを使う場合は、gradle projectをインポートしてください。

## run
gradleでbootRunタスクを実行してください。http://localhost:8080 でアクセスできます。
IDEを使って実行しても構いません。
