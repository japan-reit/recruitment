# 準備
## Java
Java11をインストールしてください。

## IDE
IDEを使う場合は、gradle projectをインポートしてください。また、[lombok](https://projectlombok.org/)を使用していますので、お使いのIDEでlombokが使えるように設定してください。

## 実行
gradleでbootRunタスクを実行してください。http://localhost:8080 でアクセスできます。
IDEを使って実行しても構いません。
